## What We Built

- **Multi-agent orchestration system** with 5 specialized agents
- **x402 payment protocol** implementation for agent-to-agent micropayments on Solana
- **Metaplex Agent Registry** integration for on-chain agent identity and discovery
- **Real-time dashboard** showing agents negotiating, bidding, and transacting live
- **Voice interface** (ElevenLabs) with text fallback for talking to agents
- **Pluggable architecture** — every integration independently enables/disables

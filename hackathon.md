### 🌐 Sovereign Infrastructure: Bittensor Challenge

**Sponsor:** Bittensor

Design a sovereign AI layer as a Bittensor subnet. The subnet owner defines work to be done and how quality is measured, then opens it permissionlessly. Distributed miners compete to produce work output; validators check quality and assign scores.

**Required Technologies:** Bittensor

**Build Requirements (all six required):**
1. **Benchmark:** target output quality metric miners are scored on
2. **Evaluation loop:** how validators score miners; cost and scale considerations
3. **Miner task:** specific, measurable, implementable task interface
4. **Incentive design:** why scoring rewards genuine quality; why top attack vectors fail
5. **Market demand:** who pays for output and why
6. **Sovereignty test:** subnet survives if any single cloud, company, or API disappears

**Developer Resources:**
- Install Bittensor SDK
- Bittensor GitHub
- Subnet Creation / Subnet Template / Vibe Codable Subnet Template

**Prizes:**

| 2nd Place | $600 USD + $1,500 Basilica AI compute credits + 1:1 mentoring and incubation |
